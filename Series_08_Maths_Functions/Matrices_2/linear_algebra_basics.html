Basics Linear Algebra

Inverting a matrix

To invert a matrix we use the command solve() with no additional argument. Remember - Not all matrices are invertible. It the determinant of a matrix is zero, then no inverse exists. 
solve(C)

##          [,1]        [,2]       [,3]
## x1  0.5714286 -0.35714286  0.2857143
## x2  0.7142857 -0.07142857 -0.1428571
## x3 -1.7142857  0.57142857  0.1428571
We can use this same command to solve a system of linear equations Ax=b. We would specify the vector b as the additional argument.
