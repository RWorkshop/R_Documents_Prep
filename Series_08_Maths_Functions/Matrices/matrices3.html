Using rbind() and cbind()

  • Another methods of creating a matrix is to "bind" a number of vectors together, either by row or by column.
  • The commands are rbind() and cbind() respectively. 
x1 <- c(1,2,4)
x2 <- c(3,8,4)
x3 <- c(1,4,3)

A <- rbind(x1,x2,x3)
B <- cbind(x1,x2,x3)

A

##    [,1] [,2] [,3]
## x1    1    2    4
## x2    3    8    4
## x3    1    4    3

B

##      x1 x2 x3
## [1,]  1  3  1
## [2,]  2  8  4
## [3,]  4  4  3

Row and Column Statistics

Statistic on the rows and columns can easily be computed if required.
  • rowMeans() : Returns vector of row means.
  • rowSums() : Returns vector of row sums.
  • colMeans() : Returns vector of column means.
  • colSums() : Returns vector of coumn means.
. 
rowMeans(A)  # Returns vector of row means.

##       x1       x2       x3 
## 2.333333 5.000000 2.666667

rowSums(A)  # Returns vector of row sums.

## x1 x2 x3 
##  7 15  8

colMeans(B)  # Returns vector of column means.

##       x1       x2       x3 
## 2.333333 5.000000 2.666667

colSums(B)  # Returns vector of coumn means.

## x1 x2 x3 
##  7 15  8
