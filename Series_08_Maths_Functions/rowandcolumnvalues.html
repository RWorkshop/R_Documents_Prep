Row and Column Statistics.

Statistic on the rows and columns can easily be computed if required. %rowMeans(A) # Returns vector of row means. %rowSums(A) # Returns vector of row sums. %colMeans(A) # Returns vector of column means. %colSums(A) # Returns vector of coumn means.
Note that the following commands are useful for Experimental Design. 


rowMeans(A)         #                                Returns vector of row means.

rowSums(A)         #                                    Returns vector of row sums. 

colMeans(A)       #                                      Returns vector of column means. 

colSums(A)        #                                       Returns vector of coumn means.
